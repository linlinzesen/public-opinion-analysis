## **一、项目概述**

### **1.1 项目目标**

设计并开发一个网络舆情事件智能分析系统，从多个信息源采集并识别热点事件，自动分析事件的传播趋势、群众情感倾向分布和风险等级，生成舆情分析报告。

## **七、本地启动说明（参考）**

### **后端启动**

cd backend
pip install flask flask-cors sqlite3
python app.py
# 服务运行于 http://localhost:5000

### **前端启动**

cd frontend
npm install
npm run dev
# 服务运行于 http://localhost:5173

### **数据采集**

cd crawler
python crawler.py
# 生成数据文件至 /data/events.json
